# Triage Labels

Engineering skills는 5개의 canonical triage role을 기준으로 말한다. 이 파일은 그 role을 이 repo issue tracker의 실제 label 문자열에 매핑한다.

| Label in mattpocock/skills | Label in our tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer가 issue를 평가해야 함 |
| `needs-info` | `needs-info` | Reporter의 추가 정보가 필요함 |
| `ready-for-agent` | `ready-for-agent` | AFK agent가 바로 처리할 수 있을 만큼 명세됨 |
| `ready-for-human` | `ready-for-human` | 사람의 직접 구현/판단이 필요함 |
| `wontfix` | `wontfix` | 처리하지 않음 |

Skill이 role을 언급하면 이 표의 오른쪽 label 문자열을 사용한다.
